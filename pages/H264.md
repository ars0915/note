- # Parse NALU（Network Abstraction Layer Unit）Header
	- H264 的 NALU
	  
	  ```
	  +------+--------------------+
	  | 0x00 | NALU Header (1B)  |
	  +------+--------------------+
	          ^
	          └─ 要解析的就是這個 byte
	  ```
	- NALU Header
	  
	  ```
	  +---------------+
	  |F| NRI |  Type |
	  +---------------+
	   1  2bits  5bits
	  ```
	  	•	F (1 bit): forbidden_zero_bit（通常為 0）
	  	•	NRI (2 bits): nal_ref_idc（重要程度，通常可忽略）
	  	•	Type (5 bits): [nal_unit_type](((675a88b9-0350-452b-87d6-0cfc9581509a)))
