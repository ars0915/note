# 時間戳與計時
	- 固定時鐘 steady clock: 系統啟動後的 ns，不能修改
	- 系統時鐘 system clock: 可修改、網路對時
	- RTCP 會定期發送 Sender Report，有每個音視頻的 timestamp 和 Network Time Protocol 時間戳，用來對齊音視頻
- # 旋轉角與朝向
